Редкие теги

Теги < dl> < dt> <dd>
Используются для удобного отображения определений

Тег <pre>
Позволяет располагать в тексте любое количество визуально определяемых "пробелов". 
 я    шел   по     улице.

Тег <code>
Позволяет отображать символы, представляющие собой программный код. Например:
function checkParent (src, dest)
Тег <gbt>
Позволяет визуально отделять часть текста, имеющую названия клавиш компьютера. Например:
Нажмите <alt> + <ctrl> + <del>

Тег <var>
Позволяет визуально отображать написание переменных компьютерных программ. Например:
При использовании Photoshop будьте внимательны к мелочам.

Теги <sup> и <sub> соответственно:
Позволяют отображать шрифты верхнего и нижнего индексов. Например:
Решим уравнение 
Х2+Y2         
Формула углекислого газа - СО2

Теги <blockquote> и <q> соответственно:
Предначзначены для выделения цитат с помощью отступов и выравниваний и с помощью кавычек. Например:
Быть человеком - самое большое в мире искусство.

и

Друзья познаются в беде.

Теги <ins> и <del>
Предназначены для зачеркивания и подчеркивания информации.
Тег <address>
Предназначен для выделения информации об авторе, - в первую очередь, для поисковых роботов, например,
этот материал был создан Ивановым Иваном

Теги <nobr> и <wbr>
1 - не допускает переносов слов. 2 - Указывает возможные места для переносов,если они потребуются.
