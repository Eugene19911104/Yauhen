Редкие теги

Теги < dl> < dt> <dd>
Используются для удобного отображения определений

Тег <pre>
Позволяет располагать в тексте любое количество визуально определяемых "пробелов". 
 я    шел   по     улице.

Тег <code>
Позволяет отображать символы, представляющие собой программный код. Например:
function checkParent (src, dest)
Тег <gbt>
Позволяет визуально отделять часть текста, имеющую названия клавиш компьютера. Например:
Нажмите <alt> + <ctrl> + <del>
